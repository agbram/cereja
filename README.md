# 🍒 Cereja

O **Cereja** é uma plataforma que conecta clientes e confeiteiras por meio de um modelo de marketplace baseado em propostas. Em vez de o cliente procurar uma confeiteira, ele publica o que deseja (bolo, doces, lembranças, kits personalizados etc.) e as confeiteiras interessadas enviam suas propostas. O cliente compara as opções e escolhe a que melhor atende às suas necessidades.

Nosso objetivo é facilitar a contratação de serviços de confeitaria, gerar mais oportunidades para confeiteiras — especialmente profissionais autônomas e iniciantes — e tornar o processo de orçamento mais simples, transparente e competitivo.

---

# 📌 Objetivo do Projeto

O Cereja busca resolver problemas encontrados tanto por clientes quanto por confeiteiras:

### Para clientes

* Encontrar diversas propostas em um único lugar.
* Comparar preços, prazos e condições.
* Economizar tempo na busca por orçamentos.

### Para confeiteiras

* Receber novos pedidos de clientes interessados.
* Aumentar a visibilidade do seu trabalho.
* Competir pela qualidade da proposta, não apenas pela divulgação.

# 🚀 Como executar o projeto

## 1. Clone o repositório

```bash
git clone https://github.com/SEU-USUARIO/cereja.git
```

## 2. Backend

```bash
cd backend
```

Execute o projeto utilizando sua IDE ou pelo Maven.

---

## 3. Frontend

```bash
cd frontend
npm install
npm run dev
```

---

## 4. Banco de Dados

Configure o MySQL localmente e ajuste as variáveis de ambiente conforme o arquivo `.env.example`.
